Help Topics

Requirements

A modern browser is required to use this web site.
  • IE8 and below will redirect to an error page
  • Some features are not supported in IE9.
  • Modern browsers include Chrome, Firefox, Opera or Safari

Accounts

An account is required to:
  • Create and edit CDEs, Forms and Boards.
  • Provide comments on CDEs
  • View some codes (i.e SNOMED)
Accounts are not needed to browse CDEs or Forms. Accounts are managed by UMLS and usually created within 24 hours.

User Profile

A registered account will be able to view his profile. In particular, he will be able to see a history of CDEs previously viewed.

All terms will be searched for. The search result will return CDEs where any of the term can be found. Priorities are given to CDEs where:
  • The search criteria is matched on name, or definition.
  • The status of the CDE is high
  • All terms are matched
  • The terms are close to one another
To force a term to be matched, it can be prepended with +
For example, 'Family +Income" will only match on CDEs that match the word income.
Quotes can also be added to ensure that terms appear as a phrase.

Roles

There are 4 roles in the CDE application.

Public User

Public users can browse the site, search and view CDES.

Registered Users

In addition to public user privileges, a registered user can create boards and add comments.

Curators

In addition to registered user privileges, a curator can create CDEs, edit CDEs owned by his steward and manage the classification tree for his steward.

Admins

In addition to curator privileges, an admin can promote/demote other users to be admin or curator for his steward.

Managing a Steward

When accounts are given administrator (admin) privilege, they are able to manage an organization. If you have an admin account, the user name menu in the upper right corner will bring down an "Account Management" option.
From the account management page, you may add other users as administrators or curators. Other administrator will also be able to add or remove administrators and curators, while curators will only be able to manage CDEs.
