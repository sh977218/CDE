The NIH Common Data Elements (CDE) Repository has been designed to provide access to structured human and machine-readable definitions of data elements that have been recommended or required by NIH Institutes and Centers and other organizations for use in research and for other purposes. Visit the NIH CDE Resource Portal for contextual information about the repository. The Repository is a platform for identifying related data elements in use across diverse areas, for harmonizing data elements,and for linking CDEs to other existing standards and terminologies, including the value sets in the Value Set Authority Center (VSAC).
