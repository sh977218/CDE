# Server(Node and Express)
## Allowed
* async
* lodash
* node
* rxjs

## Not Allowed Libraries
* angular
* webpack

## Other
* to access **server** use relative paths, such as:
   * `import {} from 'server/cde/mongo-cde';`
* to access **shared** use relative paths, such as:
   * `import {} from 'shared/item';`
